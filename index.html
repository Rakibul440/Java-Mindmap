Java Features Mindmap

Click any category to expand/collapse. Use the controls to manage all.

  • Core Language Features
    • Variables and primitive data types (int, float, double, char, boolean, byte, short, long)
    • Constants and final keyword
    • Operators (arithmetic, logical, bitwise, assignment, comparison, conditional)
    • Control flow (if/else, switch, for, while, do-while)
    • Methods & overloading
    • Arrays & multidimensional arrays
    • String handling & String pool
    • Type casting & conversion
    • Wrapper classes, Varargs, Enhanced for loop
  • Object-Oriented Programming
    • Classes & objects, Constructors & chaining, Instance vars & methods
    • Access modifiers, Inheritance (extends), Method overriding & super
    • Abstract classes & methods, Interfaces & implements, Inner classes
  • Advanced OOP Features
    • Polymorphism, Encapsulation, Abstraction, Composition, Covariant returns
    • Default/static/private methods in interfaces
  • Exception Handling
    • Try-catch-finally, throw/throws, Checked vs unchecked, Custom exceptions
    • Try-with-resources, Multi-catch, Suppressed exceptions
  • Generics and Type Safety
    • Generic classes & methods, Bounded type parameters, Wildcards, Type erasure
    • Diamond operator, Raw types
  • Collections Framework
    • List/Set/Map implementations (ArrayList, HashSet, HashMap, TreeMap...)
    • Queue/Deque, Iterators, Comparable & Comparator, Concurrent collections
  • Functional Programming (Java 8+)
    • Lambdas, Functional interfaces, Method references, Stream API, Optional, Collectors
  • Input/Output (I/O)
    • File handling, Streams, Reader/Writer, Buffered I/O, Serialization, NIO (Path/Files)
  • Multithreading and Concurrency
    • Thread/Runnable, Synchronization, Executor framework, Thread pools, Atomic classes, CompletableFuture
  • Memory Management
    • Heap vs stack, Garbage collection, Weak/Soft/Phantom references, finalize (deprecated)
  • Reflection and Introspection
    • Class loading, Runtime type info, Dynamic invocation, Annotations, Proxy classes
  • Annotations
    • Built-in annotations, Custom annotations, Retention, Target, Repeatable, Type annotations
  • Java 8 Features
    • Lambdas, Stream API, Optional, Default/static interface methods, Date/Time API
  • Java 9 Features
    • Module system, JShell, Private interface methods, Process API improvements, Reactive Streams
  • Java 10-11 Features
    • var (local type inference), HTTP Client API, String/File utility methods, Collection.toArray()
  • Java 12-17 Features
    • Switch expressions, Text blocks, Pattern matching, Records, Sealed classes, Foreign Function API
  • Java 17 Features (LTS)
    • Sealed classes finalized, Pattern matching finalized, FFM API (incubating), Enhanced PRNGs
  • Java 18+ Features
    • Pattern matching for switch (preview), Vector API, Simple web server, UTF-8 default, Virtual threads
  • Standard Library and APIs
    • java.lang, java.util, java.io, java.nio, java.net, java.sql, java.time, java.util.concurrent, streams
  • Networking
    • Sockets, URL/URLConnection, HTTP connections, Server/Datagram sockets, HTTP Client API
  • Database Connectivity
    • JDBC API, Connection/Statement/PreparedStatement, ResultSet, Drivers, Connection pooling
  • Security Features
    • Security model, SecurityManager, JCA, SSL/TLS, Access control and authentication
  • Internationalization
    • Locale, ResourceBundle, Formatting, Currency handling, Encoding support
  • Regular Expressions
    • Pattern & Matcher, Regex syntax, Group capturing and replacement
  • JVM Features
    • JIT compilation, Class loading, Bytecode, GC algorithms, JVM tuning & monitoring
  • Development Tools Integration
    • JAR/WAR, Classpath/module path, Javadoc, Unit testing, Build tools (Maven/Gradle)
